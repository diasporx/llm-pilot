# Skill: summary

Генерация саммари задачи или всего спринта. Следуй инструкции из `prompts/sprint-summary-prompt.md`.

Аргументы: `$ARGUMENTS`

Разбор аргументов:
- Если аргумент выглядит как `DEV-XXXXX` — режим одной задачи
- Если аргумент выглядит как число (например `31`) или содержит слово «спринт» — режим всего спринта

**Важно:**
- Перед обработкой любой задачи проверяй, существует ли уже файл `results/sprint-[N]/[KEY].md`. Если существует — пропускай задачу (не читай JSON, не генерируй, не перезаписывай).
- Для режима всего спринта используй Agent (subagent) для каждой задачи отдельно, чтобы не загружать основной контекст.
